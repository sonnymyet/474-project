How it works.


All the front-end components such as HTML, CSS, and Javascript from Github repository are hosted by AWS Amplify. Users make requests to the AWS Amplify for the UI components. The Javascript code contains AJAX requests to Amazon Cognito and Amazon API Gateway. Amazon Cognito manages user information such as email and password and deals with authentication. The authentication is provided by using JWT tokenization. Amazon API manages a list of Lambda functions that handles the Amazon DynamoDB operations.
